## 開発環境のバージョン管理に話題の mise を採用

```
curl http://mise.run > mise.sh
```

いきなり実行せずに curl で取得したものをスクリプトファイルとして保存し、sh で実行した。

```
sh mise.sh
```

それから、zsh 上で mise を有効化するための行を `~/.zshrc` に追記

```
echo "eval \"\$(/home/user/.local/bin/mise activate zsh)\"" >> "/home/user/.zshrc"
```

### 相対行番号表示を無効化

`~/.config/nvim/init.lua` に以下を追加 

```
vim.opt.number = true
vim.opt.relativenumber = false
```

### 日本語のスペルチェックを無効化

赤い波線が邪魔なので `~/.config/nvim/lua/config/options.lua` に以下を追加。

```
vim.opt.spelllang = { "en", "cjk" }
```

## Android の模擬に Waydroid をインストール

https://kde-diary.hatenablog.com/entry/2026/02/10/090000

```
sudo dnf install waydroid
```

waydroid 自体は、dnf でインストール。Google Play が使えるように Waydroid が起動した状態で以下のスクリプトを実行。

```
git clone https://github.com/casualsnek/waydroid_script
cd waydroid_script
python -m venv venv
venv/bin/pip install -r requirements.txt
sudo venv/bin/python3 main.py
```
