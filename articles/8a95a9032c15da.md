## Windows Terminal + PowerShell + Chafa 編

![Chafaによる画像表示](https://storage.googleapis.com/zenn-user-upload/fbc87ca9aa6a-20260302.png)

Windows Terminal バージョン 1.22 から、Sixel 画像表示をサポートしていたらしいっす。 < 何を今さら

この機能を利用すると冒頭の画像のように Windows Terminal 内に画像を表示することができます。

WSL などでの例は、いくつか見つけたのですが、Windows Native で表示する例が、あまりなかったのでメモしておきます。

## Chafa のインストール

`winget` を使用して [Chafa](https://github.com/hpjansson/chafa) をインストールします。

```
PS C:\path\to> winget install hpjansson.Chafa
見つかりました Chafa [hpjansson.Chafa] バージョン 1.18.1
このアプリケーションは所有者からライセンス供与されます。
Microsoft はサードパーティのパッケージに対して責任を負わず、ライセンスも付与しません。
インストーラーハッシュが正常に検証されました
アーカイブを展開しています...
アーカイブが正常に展開されました
パッケージのインストールを開始しています...
コマンド ライン エイリアスが追加されました: "Chafa"
インストールが完了しました
```

## Chafa による画像の表示

```
PS C:\path\to> chafa .\sample.png
```
